This is the e-commerce Prj which I've Build keeping in mind real time scenarios & Features & simplifying things for customers.
This real time WebApplication has been build inorder to have better mode of online shopping experience & exclusive AI Feature.


This is built for both user & admin, with CRUD APIs using REST service where an user can create his account, store items in cart & buy items. A subscription method introduced & based on that delivery time & charges will be modified, This is also useful for admin who can create their account and actually add their product with all details like name, quantity, price, description etc. AI Feature is also included where reviews are customized based on user behavior & previous reviews which simply filter out all 100s of reviews & generate the overall product satisfactory in simple sentence.
